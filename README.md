# CPABE Policy Builder GUI

**GUI for Policy Builder which is in turn used by CPABE - Ciphertext-Policy Attribute-Based Encryption[1] module to encrypt data**

**Pre-requisites:**  
Python 2.7  
Tkinter  
ttk  

Once you're done with the pre-requisites, run: ```python create_policy.py``` and follow the wizard instructions.

[1] http://acsc.cs.utexas.edu/cpabe/
